status - Nagios Status Objects¶

This module provides the Status class, which encapsulates a status code for Nagios.

class pynagios.status.Status(name, exit_code)[source]¶

Creates a new status object for Nagios with the given name and exit code.

Note: In general, this should never be called since the standard statuses are exported from pynagios.
