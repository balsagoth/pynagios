pynagios – Python Library for Writing Nagios Plugins¶

This package provides all the modules for writing a Nagios plugin with Python. The package file itself exports the constants used throughout the library.

pynagios.version¶

Current version of PyNagios

pynagios.PerfData¶

Alias for pynagios.perf_data.PerfData

pynagios.Plugin¶

Alias for pynagios.plugin.Plugin

pynagios.Range¶

Alias for pynagios.range.Range

pynagios.Response¶

Alias for pynagios.response.Response

pynagios.Status¶

Alias for pynagios.status.Status

pynagios.OK¶

A Status object representing the OK response status.

pynagios.WARNING¶

A Status object representing the WARNING response status.

pynagios.CRITICAL¶

A Status object representing the CRITICAL response status.

pynagios.UNKNOWN¶

A Status object representing the UNKNOWN response status.

Sub-modules:

  • perf_data – Performance Data Classes
  • plugin – Plugin Class
  • response - Response Class
  • range - Tools for Working with Nagios Ranges
  • status - Nagios Status Objects
